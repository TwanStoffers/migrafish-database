# MigraFISH

MigraFISH is a harmonised global database of freshwater migratory fishes. Version v2026.1 contains standardised information on migration classifications, migration-distance categories, confidence, conservation status, source provenance, and basin-level distributions for 2,536 species.

## Repository contents

- `code/` — scripts used to process and prepare public releases
- `shiny/` — source code for the interactive MigraFISH explorer
- `metadata/` — data dictionary and release documentation
- `documentation/` — version history

## Data access

The citable dataset will be archived on Zenodo. The DOI will be added here once the first release is published.

## Licence

The MigraFISH dataset will be released under CC BY 4.0. This repository contains code and documentation supporting the database; raw third-party source data are not redistributed here because some sources have their own access and reuse conditions.
