# MigraFISH v2026.1

## Global database of freshwater migratory fishes

MigraFISH is a harmonised global database of freshwater fish species known or thought to exhibit migratory behaviour. Version 2026.1 contains 2,536 species and integrates standardised information on migration class, maximum migration-distance category, confidence, population-level variation in migratory behaviour, conservation status, source provenance, and basin-level distributions.

## Files in this release

- **MigraFISH_v2026.1.csv** — the canonical machine-readable dataset. Use this file for analysis.
- **MigraFISH_v2026.1.xlsx** — a spreadsheet version of the same dataset for browsing and manual inspection.
- **MigraFISH_data_dictionary_v2026.1.xlsx** — definitions, formats, allowed values, sources, and caveats for all fields.
- **MigraFISH_version_history.csv** — release history (if included in this archive).

## Scope and interpretation

MigraFISH includes species that undertake predictable and ecologically meaningful movements involving freshwater habitats during their life cycle. This includes potamodromous and diadromous species. `Coastal` and `Non-migratory` records are retained where one or more input sources identified a species as migratory but the review could not verify it as a freshwater migratory fish. `Little information` indicates that available evidence was insufficient to assign a migration class confidently.

Basin-level distributions are represented by **Pfafstetter level-5 basin codes** derived from the HydroBASINS framework. These fields represent documented occurrence or assessment coverage from the underlying spatial sources and should not be interpreted as complete occupancy of every river reach within a basin.

## Versioning

MigraFISH is maintained through versioned releases. The Zenodo concept DOI will resolve to the latest release, whereas the version DOI identifies the exact archived dataset used in an analysis. Previous versions remain available for reproducibility.

## Contact and updates

Users can submit corrections, supporting references, taxonomic updates, revised migration classifications, or additional species through the MigraFISH Shiny application or the project repository. Submitted information is reviewed before incorporation into a future version.
