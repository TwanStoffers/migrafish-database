# Processing scripts

This folder contains the R scripts used to prepare MigraFISH spatial data and create versioned public releases.

## Scripts

- `01_prepare_spatial_data.R` — harmonises spatial information and prepares the processed MigraFISH database.
- `02_create_release_files.R` — creates the versioned public release files and performs release checks.

## Data inputs

The scripts use local input data that are not included in this repository. These include source-derived spatial inputs and other third-party data products that may have their own access and reuse conditions. File paths and required inputs are documented within the scripts.
